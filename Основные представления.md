**Меры:**

1. Шифрование:

  - TLS 1.3 (все коммуникации)
  - AES-256 (данные в rest)

2. Аутентификация:

  - JWT с 15-минутным TTL
  - OAuth 2.0 для внешних API

3. Мониторинг угроз:

  - SIEM (Splunk)
  - Аномалии доступа (AWS GuardDuty)
